# QA-Z v0.9.8-alpha

QA-Z `v0.9.8-alpha` is a Codex-first, model-agnostic QA control plane alpha for coding-agent workflows. The matching Python package version is `0.9.8a0`.

## Included

- Deterministic `init`, `plan`, `fast`, `deep`, `review`, and `repair-prompt` command surfaces remain the core contract.
- Fast and deep QA gates produce repairable evidence through local run artifacts, SARIF, GitHub summary output, and post-repair verification.
- Benchmark fixtures cover Python, TypeScript, mixed fast/deep handoff aggregation, executor bridge action context, executor-result ingest, and the live-free safety dry-run.
- The root repository gate uses a Python-only `qa-z.yaml`; TypeScript coverage remains in `qa-z.yaml.example`, the demo, and benchmark fixtures.

## Not Included

- QA-Z does not perform live repair or call Codex, Claude, or another agent automatically in this alpha.
- Deep QA remains deterministic Semgrep-backed execution, not an LLM-only judgment layer.
- Generated runtime artifacts under `.qa-z/**` and `benchmarks/results/**` stay local unless a release commit explicitly freezes evidence with context.
